# zrender
+ 是一个轻量级的2d js图形库，echarts的底层渲染引擎

## ZRender
+ zrender的核心类，管理画布的初始化、管理图形元素、事件监听、画布销毁。包括4个组件：
1. Storage：存储和管理图形元素
2. painter：负责渲染，有canvas和svg两种实现
3. Handler：负责事件交互
4. Animation：负责动画

主要功能

```javascript
add(el: Element)      // 添加元素
remove(el: Element)   // 删除元素
clear()               // 清空所有元素
```

```javascript
refresh()             // 刷新画布，通过帧请求函数requestAnimationFrame来刷新，最终还是调用了refreshImmediately
refreshImmediately()  // 立即刷新，调用painter.refresh刷新画布
```

```javascript
on(eventName, handler)    // 绑定事件，通过handler.on实现
off(eventName, handler)   // 解绑事件,通过handler.off实现
trigger(eventName, event) // 触发事件,通过handler.trigger实现
```

## painter
+ 负责渲染图形元素，将图形绘制到画布，有2个实现：canvas和svg
+ 支持分层绘制，根据图形元素的zlevel值，将图形元素分到对应的图层(getLayer(zlevel))

核心属性

```javascript
storage: Storage                  // 存储管理器
private _layers: {[key: number]: Layer}  // 图层管理
private _zlevelList: number[]     // zlevel 层级列表
```

渲染控制

```javascript
refresh(paintAll?: boolean)       // 刷新画布，从storage中获取所有的图形元素，清空画布，重绘
refreshHover()                    // 刷新悬停层
paintOne(ctx, el)  
```

refresh流程

```javascript
refresh
  _paintList(list: Displayable[]...) //list是storage中获取的图形元素
    _updateLayerStatus//根据图形元素的zlevel创建图层
    _doPaintList//遍历图层，遍历元素
      _doPaintEl//绘制单个元素
```

脏矩形优化

+ 只重绘变化的矩形区域

```javascript
layer.createRepaintRects //_doPaintList遍历图层时,计算当前图层的脏矩形，当开启脏矩形优化时，每次更新图形元素时，会和更新前的图形元素进行对比，如果发生新增、删除、修改(图形元素属性修改时，dirty脏位会置位)，则将变化的图形元素所在的矩形区域合并到已有的脏矩形区域。重绘时，通过canvasContext.clearRect(x, y, width, height)只清空脏矩形区域，ctx.rect(x,y,width, height)和ctx.clip只绘制脏矩形区域
```

## Animation
+ 是动画系统的核心控制器，通过动画切片Clip链表来管理和调度动画

核心属性

```javascript
class Animation extends Eventful {
    stage: Stage                    // 舞台，用于触发渲染更新
    private _head: Clip            // 动画片段链表头
    private _tail: Clip            // 动画片段链表尾
    private _running: boolean      // 是否正在运行
    private _time: number          // 当前时间
    private _paused: boolean       // 是否暂停
}
```

动画片段管理

```javascript
addClip(clip: Clip)           // 添加动画片段
removeClip(clip: Clip)        // 移除动画片段
addAnimator(animator: Animator) // 添加动画控制器
removeAnimator(animator: Animator) // 移除动画控制器
```

动画生命周期

```javascript
start()     // 启动动画循环
stop()      // 停止动画
pause()     // 暂停动画
resume()    // 恢复动画
clear()     // 清空所有动画
```

动画执行

```javascript
//通过requestAnimationFrame设置帧的回调方法
_startLoop
/*
帧的回调方法
遍历切片链表，依次调用Clip.step，执行每个图形元素的一帧动画，其实就是更新每个图形元素的一帧动画对应的属性值
调用stage.update，读取图形元素更新后的属性值，重新渲染
*/
update(notTriggerFrameAndStageUpdate?: boolean)
```

## Animator
+ 代表单个图形对象的动画，负责创建动画轨道Track、创建动画切片Clip，添加切片到Animation

属性

```javascript
animation?: Animation //对应的动画对象
private _tracks: Dictionary<Track> = {}
private _trackKeys: string[] = []
private _target: T //图形对象
private _clip: Clip = null
```

方法

```javascript
/*
给props中的属性，添加2个关键帧，分别是0时刻和time时刻
*/
whenWithKeys(time: number, props: Dictionary<any>, propNames: string[], easing?: AnimationEasing)
/*
遍历tracks,调用prepare
创建动画切片Clip，设置Clip.onFrame回调，调用所有轨道的step，添加进Animation
*/
start(easing?: AnimationEasing)
```

## Track
+ 代表一个图形元素的一个属性的动画，由多个关键帧组成，每个关键帧对应动画的一个时刻，其中保存了那个时刻的属性值

属性

```javascript
keyframes: Keyframe[] = []      // 关键帧数组
propName: string                // 属性名（如 'x', 'y', 'scale'）
private _lastFr = 0 //最近一次执行的关键帧
```

关键帧

```javascript
type Keyframe = {
    time: number                    // 时间点（毫秒）
    value: unknown                  // 插值后的值
    percent: number                 // 时间百分比
    rawValue: unknown              // 原始值（用于离散动画）
    easing?: AnimationEasing       // 缓动函数
    easingFunc?: (percent: number) => number  // 缓动函数实例
    additiveValue?: unknown        // 叠加动画值
}
```

方法

```javascript
/*
添加关键帧
time: 关键帧对应的时刻，0~duration,duration是动画持续时间
rawValue：关键帧对应的属性值
*/
addKeyframe(time: number, rawValue: unknown, easing?: AnimationEasing)
/*
计算关键帧中的percent，percent=time/maxTime，帧的时刻/动画持续时间
*/
prepare(maxTime: number, additiveTrack?: Track)
/*
从<=percent的关键帧frame开始，执行一个关键帧，其实就是更新图形元素的属性
percent是当前动画的全局进度(0-1)
w=(percent - frame.percent)/(nextFrame.percent-frame.percent)，当前动画在frame到nextFrame之间的相对进度
属性更新值=frame.value+w*(nextFrame.value - frame.value)
*/
step(target: any, percent: number)
```

## Clip
+ 动画切片，代表一个图形对象的当前动画的执行，负责执行当前动画帧，包含了一个图形对象所有轨道的当前帧

属性

```javascript
onframe: OnframeCallback //当前帧动画的回调函数
private _startTime = 0 //动画的开始时间
private _life: number //动画的总时间
```

执行当前帧

```javascript
/*
计算动画进度percent，(getTime()-_startTime)/_life
调用onframe(percent)
*/
step(globalTime: number, deltaTime: number)
```

## 核心流程
创建动画

```javascript
/*
图形元素的animateTo函数，比如：
animateCircle.animateTo({
            shape: { r: 40 },
            style: { fill: 'green' }
        }, {
            duration: 1000,
            easing: 'bounceOut'
        });
target：目标属性
cfg: 动画配置
*/
Element.animateTo(target: Props, cfg?: ElementAnimateConfig, animationProps?: MapToType<Props, boolean>) {
  animateTo(this, target, cfg, animationProps);
}
```

调用链

```javascript
Element.animateTo
  Element.animateTo
    Element.animateToShallow//遍历动画要修改的属性，创建Animator
      animator.whenWithKeys
    animator.start
```

# Echarts
## Echarts
+ 是Echarts的核心类，提供了外部程序使用的接口。使用了分层渲染架构：
1. 数据层: 处理原始数据
2. 模型层: 管理图表配置和状态
3. 视图层: 负责图形渲染
4. 交互层: 处理用户交互
+ 模型层和视图层分为chartsModel、chartsView和componentModel、componentView，charts表示series选项对应的图表，是频繁变化的数据，component表示非series选项对应的组件，不频繁变化

属性

```javascript
_zr: zrender.ZRenderType; //底层的zrender实例
_dom: HTMLElement;//zrender的根dom元素
_model: GlobalModel;//全局模型
_chartsViews: ChartView[] = []; //series选项对应的视图，图表视图
_chartsMap: {[viewId: string]: ChartView} = {};
_componentsViews: ComponentView[] = []; //非series选项对应的视图，组件视图
_componentsMap: {[viewId: string]: ComponentView} = {};
```

方法

```javascript
//静态方法，echarts实例的工厂方法，会创建并返回echarts实例，将实例保存到全局属性instances
export function init(
    dom?: HTMLElement | null,
    theme?: string | object | null,
    opts?: EChartsInitOpts
)
/*
配置echarts，配置完成后，图表会显示到页面上
如果是第一次调用，会先创建并初始化全局模型GlobalModel
调用prepare，创建视图，分为chartsView和componentView，根据选项的key,创建和初始化视图对象XXView，将XXView.group添加到zrender，group是zrender的group，后续会添加该视图对应的zrender图形实例
执行XXView.render，将视图组件渲染到页面
*/
setOption<Opt extends ECBasicOption>(option: Opt, notMerge?: boolean | SetOptionOpts, lazyUpdate?: boolean): void
```

属性

方法

